# DONE_CHECKLIST.md

개발 완료 전에 아래를 반드시 확인:

## A. 기능/디자인
- [ ] PDF 원본 대비 레이아웃이 일치하는가?
- [ ] 임의 리디자인 요소가 없는가?
- [ ] 반응형/기본 인터랙션 동작이 정상인가?

## B. 품질
- [ ] `npm run lint` 통과
- [ ] `npm run typecheck` 통과 (지원 시)
- [ ] 콘솔 에러/경고가 없는가?

## C. 문서/리뷰
- [ ] 변경 파일/핵심 포인트를 PR 설명에 작성했는가?
- [ ] 전/후 스크린샷 첨부했는가?
- [ ] 리스크/추가 작업 TODO를 적었는가?

## D. Git
- [ ] 커밋 메시지가 규칙(`feat/fix/chore(scope): ...`)에 맞는가?
- [ ] 불필요 파일(.DS_Store, 임시 파일 등) 제외했는가?
- [ ] 대상 브랜치가 올바른가?
