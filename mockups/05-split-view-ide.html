실습 1: 정렬 알고리즘 구현

문제 설명

주어진 배열을 버블 정렬 알고리즘을 사용하여 오름차순으로 정렬하는 함수 bubble_sort를 구현하세요.

입력

정수로 이루어진 리스트 arr (1 ≤ len(arr) ≤ 1000)

출력

오름차순으로 정렬된 리스트

예시

# 입력
arr = [64, 34, 25, 12, 22, 11, 90]
# 출력
[11, 12, 22, 25, 34, 64, 90]

힌트

버블 정렬은 인접한 두 원소를 비교하여 정렬하는 방식입니다. 이중 반복문을 사용하세요.

solution.py
1 def bubble_sort(arr):
2 # 여기에 코드를 작성하세요
3 n = len(arr)
4 for i in range(n):
5 for j in range(0, n-i-1):
6 if arr[j] > arr[j+1]:
7 arr[j], arr[j+1] = arr[j+1], arr[j]
